# student grademanagment system
Description
The Student Grading System is a command-line based application developed using Python and SQLite. It includes user authentication and allows authenticated users to manage students, subjects, and grades. Features

Requirements
Python 3.x
SQLite3
Any Python IDE (VS Code / PyCharm)
How to Run the Project
Download or clone the repository
Open terminal in the project folder
Run the following command:
Register a new user
Login using username and password
Use menu options to manage student grading system
Database
The database file 'students.db` is created automatically
Data is stored permanently using SQLite
